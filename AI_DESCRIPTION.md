# รายละเอียด AI ที่ใช้ในโครงงาน

## โครงงาน: StyleSpace — ระบบออกแบบตกแต่งภายในด้วย AI

---

## 1. ประเภทของ AI ที่ใช้

| ประเภท | การนำไปใช้ในโครงงาน |
|---|---|
| **Multimodal AI (Vision-Language Model)** | ใช้ Claude Vision วิเคราะห์ภาพแปลนห้อง — รับ Input เป็นภาพ + ข้อความ แล้วตรวจจับตำแหน่ง ขนาด และประเภทของแต่ละห้องโดยอัตโนมัติ ส่งผลลัพธ์เป็น Structured JSON (ไม่ใช่ Computer Vision แบบดั้งเดิมที่ใช้ CNN/YOLO แต่เป็น Large Language Model ที่มีความสามารถในการเข้าใจภาพ) |
| **Generative AI (Image Generation)** | ใช้ Google Gemini สร้างภาพ Photorealistic Isometric 3D Render จากแปลนห้อง 2D ในสไตล์การตกแต่งภายในที่ผู้ใช้เลือก รวมถึง Restyle ภาพ Render เดิมเป็นสไตล์ใหม่ |
| **Prompt Engineering** | แปลงคำตอบจากแบบสอบถาม 6 ข้อ ให้เป็น Prompt ภาษาอังกฤษที่เหมาะสม โดยรวม Prompt Tags จากแต่ละตัวเลือกเข้าด้วยกัน เพื่อสร้างคำสั่งที่เฉพาะเจาะจงสำหรับ Gemini Image Generation |

## 2. เครื่องมือและแพลตฟอร์มที่ใช้

### AI Models

| เครื่องมือ | Model | บทบาท |
|---|---|---|
| **Anthropic Claude** | `claude-sonnet-4-20250514` | Vision-Language Model — รับภาพแปลนห้อง + ข้อความสั่งงาน แล้วตรวจจับห้อง ระบุประเภท คำนวณตำแหน่งและขนาดเป็น Bounding Box (JSON) |
| **Google Gemini** | `gemini-2.5-flash-image` | สร้างภาพ Render จากแปลนห้อง + Style Prompt และ Restyle ภาพ Render เดิมเป็นสไตล์ใหม่ |

### Tech Stack

| เทคโนโลยี | การใช้งาน |
|---|---|
| **FastAPI (Python)** | Backend API server — รับภาพจาก Frontend ส่งต่อไป Claude/Gemini แล้วส่งผลลัพธ์กลับ |
| **Anthropic Python SDK** | เชื่อมต่อ Claude Vision API สำหรับวิเคราะห์แปลนห้อง |
| **Google GenAI SDK** | เชื่อมต่อ Gemini Image Generation API สำหรับสร้างและ Restyle ภาพ Render |
| **Pillow (Python)** | ปรับขนาดภาพ Input ก่อนส่งไป AI เพื่อลดต้นทุน Token |
| **Vite + TypeScript** | Frontend — UI สำหรับอัปโหลด ทำแบบสอบถาม เลือกสไตล์ และดูผลลัพธ์ |

## 3. บทบาทของ AI ในโครงงาน

### ปัญหาที่แก้ไข

ปกติการออกแบบตกแต่งภายในต้องจ้าง Interior Designer ซึ่งมีค่าใช้จ่ายสูงและใช้เวลานาน ผู้ใช้ทั่วไปไม่สามารถจินตนาการได้ว่าห้องของตัวเองจะดูเป็นอย่างไรในสไตล์ต่าง ๆ ก่อนตัดสินใจซื้อวัสดุ

### วิธีการทำงานของ AI (Pipeline)

```
[ผู้ใช้อัปโหลดแปลนห้อง]
         │
         ▼
┌─────────────────────────────────┐
│  Claude Vision (Computer Vision) │ ← วิเคราะห์แปลนห้อง
│  - ตรวจจับจำนวนห้อง              │    ระบุประเภท (ห้องนอน, ครัว, ฯลฯ)
│  - คำนวณตำแหน่งและขนาดแต่ละห้อง   │    ส่งกลับเป็น JSON
└─────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────┐
│  แบบสอบถาม 6 ข้อ (Quiz)         │ ← ถามความชอบของผู้ใช้
│  - บรรยากาศที่ต้องการ             │    เช่น สว่าง / เงียบสงบ / เท่
│  - สีที่ชอบ                      │    เช่น ขาว / เบจ / เข้ม
│  - พื้นผิววัสดุ                   │    เช่น ไม้ / คอนกรีต / หินอ่อน
│  - รูปแบบเฟอร์นิเจอร์             │
│  - สไตล์การตกแต่ง                 │
│  - กิจกรรมหลักในบ้าน              │
│                                 │
│  → คำนวณ Score → จัดอันดับสไตล์  │
│  → สร้าง Custom Prompt จาก Tags  │
└─────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────┐
│  Gemini Image Generation         │ ← สร้างภาพ Render
│  - รับแปลนห้อง + ข้อมูลห้อง       │
│    + Style Prompt                │
│  - สร้างภาพ Isometric 3D         │
│    แบบ Photorealistic            │
└─────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────┐
│  Gemini Restyle                  │ ← เปลี่ยนสไตล์
│  - รับภาพ Render เดิม            │    ไม่ต้องอัปโหลดแปลนใหม่
│    + Style Prompt ใหม่           │
│  - รักษา Layout เดิม             │
│    เปลี่ยนเฉพาะเฟอร์นิเจอร์/วัสดุ  │

         ▼
[ผู้ใช้เห็นภาพ Render ในสไตล์ต่าง ๆ ได้ทันที]
```

### AI ทำงาน 2 ขั้นตอนหลัก

1. **วิเคราะห์แปลนห้อง (Claude Vision)**
   - รับภาพแปลนห้อง 2D เป็น Input
   - ใช้ Computer Vision ตรวจจับห้องแต่ละห้อง
   - ส่งกลับ JSON ที่มี: ชื่อห้อง, ประเภท (living/bedroom/kitchen/bathroom/dining/corridor/balcony), ตำแหน่ง x,y และขนาด width,depth (เป็นสัดส่วน 0.0–1.0)
   - มีระบบ Auto-fix Overlap ป้องกันห้องทับซ้อนกัน

2. **สร้างภาพ Render (Gemini Image Generation)**
   - **Base Render**: รับแปลนห้อง + ข้อมูลห้องจาก Claude + Style Prompt → สร้างภาพ Isometric 3D ครั้งแรก
   - **Restyle**: รับภาพ Render เดิม + Style Prompt ใหม่ → เปลี่ยนสไตล์โดยรักษา Layout เดิม (เร็วกว่าและสม่ำเสมอกว่าการ Render ใหม่ทั้งหมด)

## 4. จุดเด่นและนวัตกรรม

### 4.1 แบบสอบถามอัจฉริยะ (Intelligent Quiz System)

- แบบสอบถาม 6 ข้อ ใช้ภาษาที่เข้าใจง่าย ไม่ใช้ศัพท์เทคนิค
- แต่ละตัวเลือกมี **Weighted Score** ที่สัมพันธ์กับ 6 สไตล์การตกแต่ง
- แต่ละตัวเลือกมี **Prompt Tags** (เช่น "warm earth tone palette", "exposed concrete walls") ที่จะถูกรวมกันเป็น Custom Prompt สำหรับ Gemini
- ผลลัพธ์: ผู้ใช้ได้สไตล์ที่ออกแบบเฉพาะตัว + สไตล์สำเร็จรูปเรียงตาม % ความตรง

### 4.2 ลดต้นทุน API อย่างมีประสิทธิภาพ

| เทคนิค | ผลลัพธ์ |
|---|---|
| **Image Resize (512px)** | ลดขนาดภาพ Input ก่อนส่งไป Gemini ด้วย Pillow → ลด Input Token ~75% |
| **Server-side Cache** | เก็บผลลัพธ์ Render ในไฟล์ โดย Hash จากภาพ + สไตล์ → แปลนเดิม + สไตล์เดิม ไม่เรียก API ซ้ำ |
| **Restyle แทน Re-render** | เปลี่ยนสไตล์โดยใช้ภาพ Render เดิมเป็น Base → ไม่ต้องส่งแปลนห้องซ้ำ |
| **Frontend Render Cache** | เก็บภาพ Render ที่ได้รับไว้ใน Memory → สลับดูสไตล์ที่เคยสร้างแล้วได้ทันที |

### 4.3 รองรับ 2 ภาษา (Thai / English)

- UI ทั้งหมดรองรับภาษาไทยและอังกฤษ สลับได้แบบ Real-time
- แบบสอบถามมีทั้ง 2 ภาษา แต่ Prompt Tags เป็นภาษาอังกฤษเสมอ (เพื่อให้ Gemini เข้าใจได้ดีที่สุด)

### 4.4 สไตล์ที่หลากหลาย ตอบโจทย์ตลาดไทย

| สไตล์ | คำอธิบาย |
|---|---|
| Modern Minimal | เรียบง่าย โทนสีกลาง พื้นที่โปร่ง |
| Japanese Zen | วัสดุธรรมชาติ โทนดิน สงบเป็นหนึ่ง |
| Industrial Loft | คอนกรีตดิบ โครงสร้างเปิด สไตล์เมือง |
| Scandinavian | สว่าง อบอุ่น ไม้สีอ่อน ใช้งานได้จริง |
| Thai Contemporary | ไม้สักอบอุ่น ดินเผา หรูแบบเขตร้อน |
| Luxury Modern | หินอ่อน โทนเข้ม ดีเทลทอง พรีเมียม |

## 5. ประโยชน์เชิงใช้งานจริง (Practical Impact)

### สำหรับผู้ใช้ทั่วไป
- **ลดเวลา**: จากเดิมต้องรอ Interior Designer ทำแบบหลายวัน → เห็นภาพ Render ได้ภายในไม่กี่วินาที
- **ลดค่าใช้จ่าย**: ไม่ต้องจ้างนักออกแบบเพื่อดูตัวอย่างสไตล์ → ทดลองได้ฟรีไม่จำกัดสไตล์
- **ตัดสินใจง่ายขึ้น**: เห็นภาพจริงของห้องตัวเองในหลายสไตล์ → เปรียบเทียบได้ก่อนซื้อวัสดุ
- **ไม่ต้องมีความรู้ด้านออกแบบ**: แบบสอบถามช่วยค้นหาสไตล์ที่เหมาะกับไลฟ์สไตล์โดยไม่ต้องรู้ศัพท์เทคนิค

### สำหรับ SCG (ธุรกิจ)
- เป็นเครื่องมือช่วยขายที่แสดงสินค้า SCG ในบริบทจริงของห้องลูกค้า
- สร้างประสบการณ์ใหม่ที่เชื่อมโยง แปลนห้อง → ภาพตกแต่ง → สินค้า SCG ที่เหมาะสม

### โอกาสต่อยอดในอนาคต
- **เพิ่ม 3D Interactive Viewer**: ให้ผู้ใช้หมุนดูห้องจากมุมต่าง ๆ ด้วย Three.js
- **เชื่อมต่อ E-commerce**: คลิกสินค้าในภาพ Render → ลิงก์ไปหน้าสั่งซื้อ SCG โดยตรง
- **รองรับ AR/VR**: ใช้ภาพ Render เป็น Texture ใน AR experience
- **AI ประมาณราคา**: คำนวณค่าใช้จ่ายวัสดุจากขนาดห้องและสไตล์ที่เลือก
- **รองรับแปลนห้องหลายชั้น**: วิเคราะห์แปลนหลายชั้นพร้อมกัน
